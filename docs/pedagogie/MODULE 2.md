# MODULE 2

MODULE 2 — DOCUMENT DE CONCEPTION DÉTAILLÉ

Fonction du module, choix visuels, logique pédagogique, interséances

1. PLACE DU MODULE 2 DANS LA MÉTHODE

Le Module 2 arrive juste après le Module 1.

Le Module 1 a installé une idée essentielle :

une image ne raconte jamais une seule histoire.

Les élèves y découvrent que :
leur regard a de la valeur,
ils peuvent interpréter,
ils peuvent imaginer,
deux élèves peuvent voir des choses très différentes à partir de la même image.

Mais à la fin du Module 1, les élèves sont encore dans une forme de liberté brute.
Ils sentent qu’ils peuvent imaginer, mais ils ne savent pas encore ce qui fait tenir une scène.

Le Module 2 sert donc à faire basculer l’élève :
du simple imaginaire
vers la compréhension d’un mécanisme.

Autrement dit, le Module 2 est le moment où l’on commence à faire comprendre que :

un film n’est pas une accumulation d’idées ou d’actions.
Un film est une suite de choix sous contrainte.

C’est le premier module où Banlieuwood commence à introduire la mécanique invisible de la dramaturgie.

2. OBJECTIF PROFOND DU MODULE 2

Le but du Module 2 n’est pas d’apprendre à écrire.

Le but est de provoquer une prise de conscience :

Une scène n’existe pas parce qu’il se passe beaucoup de choses.
Une scène existe parce que quelqu’un veut quelque chose, rencontre un obstacle, et qu’il se produit un changement.

Cette idée doit être ressentie, pas seulement comprise intellectuellement.

C’est pour cette raison que le module ne prend pas la forme d’un cours, mais d’une expérience sous contrainte.

3. CE QUE LE MODULE 2 DOIT PRODUIRE CHEZ L’ÉLÈVE

À la fin du module, l’élève n’est pas censé savoir théoriser la dramaturgie.

En revanche, il doit avoir ressenti que :
une scène a besoin d’une intention,
une scène a besoin d’un problème,
une scène a besoin d’un changement,
on ne peut pas tout mettre,
choisir, c’est déjà écrire.

C’est une différence majeure.

Le module ne cherche donc pas à faire “apprendre une notion”.
Il cherche à faire sentir une mécanique.

4. POURQUOI LE MODULE 2 EST CONÇU EN DEUX BLOCS

Le Module 2 a été pensé en deux blocs complémentaires :

Bloc A — Cinéma & lecture

Un bloc d’entrée, de mise en bain, qui prend appui sur des références que les élèves connaissent.

Bloc B — Jeu principal “Émotion cachée”

Un bloc de mise en pratique profonde, où ils expérimentent eux-mêmes les contraintes d’écriture.

Pourquoi ce double mouvement ?

Parce que si on attaque directement par le jeu, certains élèves ne comprennent pas ce qu’on cherche à leur faire ressentir.
Et si on reste seulement dans l’analyse de scènes, on tombe dans quelque chose de trop scolaire.

Le bloc A sert donc à ouvrir le cerveau.
Le bloc B sert à faire vivre la mécanique.

5. BLOC A — CINÉMA & LECTURE

Pourquoi il existe

Ce bloc a trois fonctions.

Fonction 1 — Créer une base commune

Tous les élèves n’ont pas la même culture cinéma.

Certains ont vu beaucoup d’œuvres.
D’autres presque aucune.
D’autres encore consomment surtout des contenus courts, YouTube, TikTok ou des animés.

Si on parle de “scène forte” sans point d’appui, on perd la moitié de la classe.

Le bloc A permet donc de partir de ce qu’ils connaissent déjà.

Fonction 2 — Faire un pont entre spectateur et créateur

Avant d’inventer eux-mêmes, les élèves doivent comprendre que les scènes qui les marquent ont une structure.

Ils ont tous déjà ressenti qu’une scène était forte.

Le bloc A sert à leur faire prendre conscience que ce ressenti n’est pas magique.

Fonction 3 — Introduire les notions sans les nommer brutalement

Le module veut faire comprendre :
intention
obstacle
changement

Mais si on arrive frontalement avec ces mots, on casse le côté ludique.

Le bloc A permet d’introduire ces notions à partir de scènes connues, sans donner l’impression qu’on est dans un cours.

6. VISUEL DU BLOC A — CE QUI A ÉTÉ CHOISI

Le bloc A doit être visuellement simple, lisible, immédiatement compréhensible.

Nous avons donc choisi :

A1 — Une interface de sélection de contenus connus

Les élèves voient une liste de contenus avec visuels :
films,
séries,
animés,
références narratives grand public.

Ils peuvent en cocher plusieurs.

Pourquoi cette forme visuelle ?

Parce qu’elle est immédiatement familière.
On n’est pas dans une question abstraite du type :

“Quelles sont vos références culturelles ?”

On est dans quelque chose de concret :

“Coche ce que tu connais.”

C’est rapide, peu intimidant, et ça permet de créer une cartographie de la culture commune de la classe.

Pourquoi une liste fermée et pas une zone libre ?

Parce qu’on veut :
éviter les fautes,
éviter les réponses introuvables,
garder des références maîtrisées par Banlieuwood,
pouvoir réutiliser les résultats de manière structurée.

Donc ici, le visuel n’est pas juste pratique :
il sert directement la logique de collecte de données.

A2 — Une zone de texte libre pour raconter une scène marquante

Une fois qu’un élève choisit un contenu, il doit raconter une scène qui l’a marqué.

Le visuel ici doit être très sobre :
le titre de l’œuvre,
une zone de texte,
une consigne courte.

Pourquoi une zone de texte libre et pas un QCM ?

Parce qu’à ce stade, on veut voir comment l’élève raconte spontanément.

C’est une observation précieuse :
certains décrivent juste les actions,
certains parlent des émotions,
certains identifient déjà les enjeux.

Le texte libre permet donc de voir le “réflexe narratif brut”.

A3 — Une deuxième question “entre les lignes”

Créer un système de contrôle de la matière. C a d choisir des scènes fortes de tous les films présentés. Afin que l’on puisse cadrer. C’est, cependant, un gros travail de recherche. Mais qui peut être accéléré par l’ia. 

Ensuite, une nouvelle question apparaît, plus profonde :

Dans cette scène, qu’est-ce qui se passe vraiment ?
Qu’est-ce que le personnage veut ?
Ou qu’est-ce qui change pour lui ?

Visuellement, on reste dans le même système, mais la consigne change de niveau.

Pourquoi ce choix ?

Parce qu’on veut forcer un déplacement mental.

La première question demandait :

qu’est-ce qui t’a marqué ?

La seconde demande :

qu’est-ce qui structure la scène ?

On crée ainsi une montée invisible en complexité.

7. POURQUOI ON N’A PAS FAIT UN “COURS DE CINÉMA” À CE MOMENT-LÀ

C’était une possibilité.

On aurait pu imaginer un bloc où l’intervenant explique :
ce qu’est une intention,
ce qu’est un obstacle,
ce qu’est un changement.

Mais on l’a écarté.

Pourquoi ?

Parce que ce serait beaucoup trop tôt.

À ce stade, les élèves n’ont pas encore besoin d’un vocabulaire théorique.
Ils ont besoin d’une expérience de lecture.

Si on théorise trop tôt :
on perd la spontanéité,
on donne l’impression qu’il y a une bonne réponse,
on scolaire un moment qui doit rester ouvert.

8. TRANSITION ENTRE LE BLOC A ET LE BLOC B

À ce moment du module, l’intervenant doit faire une transition très simple.

Pas un long discours.

Le message à faire passer est :

Dans une scène, il ne suffit pas qu’il se passe quelque chose.
Il faut que quelqu’un veuille quelque chose, qu’il y ait un problème, et qu’il se passe un vrai changement.

C’est le moment où Banlieuwood commence à faire sentir que :

l’écriture, ce n’est pas juste “avoir une idée”.
C’est organiser des tensions.

9. BLOC B — LE JEU “ÉMOTION CACHÉE”

Pourquoi il est le cœur du module

Le bloc B est le véritable moteur du Module 2.

C’est là que les élèves expérimentent eux-mêmes la mécanique dramaturgique.

Le jeu a été pensé pour provoquer plusieurs prises de conscience en même temps.

1. Une scène part souvent d’une émotion, pas d’un événement spectaculaire.

2. Une scène a besoin d’une structure.

3. Tout ne peut pas rentrer dans une scène.

4. Plus on ajoute des éléments, plus il faut justifier.

5. Les contraintes obligent à penser.

10. POURQUOI LE JEU COMMENCE PAR UNE ÉMOTION

On a choisi de faire partir le jeu d’une émotion contextualisée.

Les émotions choisies sont :
peur d’être exclu,
colère face à une injustice,
honte après une erreur,
jalousie envers un ami,
joie fragile.

Pourquoi ce choix est très important

Parce qu’il ancre le jeu dans quelque chose que les collégiens connaissent intimement.

On ne leur demande pas :

invente une scène de cinéma.

On leur demande, sans le dire frontalement :

pars de quelque chose que tu peux ressentir.

Cela change tout.

Un enfant peut ne pas se sentir “créatif”, mais il sait ce que c’est :
avoir honte,
être jaloux,
avoir peur d’être exclu.

Donc on sécurise l’entrée dans le jeu.

11. POURQUOI CES ÉMOTIONS ET PAS D’AUTRES

Nous avons choisi des émotions :
fortes mais quotidiennes,
dramatiques mais non traumatiques,
adaptées au cadre scolaire.

On a évité :
la violence lourde,
les émotions trop abstraites,
les choses trop éloignées de leur quotidien.

Le module doit rester proche de la vie réelle du collège.

12. VISUEL DU JEU “ÉMOTION CACHÉE”

Le visuel du jeu doit immédiatement faire comprendre que :
ce n’est pas un cours,
ce n’est pas seulement une question/réponse,
c’est une scène à construire.

Le cœur visuel du module repose sur deux zones.

Zone A — La structure narrative visible en permanence

Trois blocs toujours visibles :
Mon personnage veut…
Mais…
À la fin de la scène…

Pourquoi cette structure visible tout le temps ?

Parce qu’on veut que l’élève voie en permanence la charpente d’une scène.

Ce n’est pas un champ qu’on remplit puis qu’on oublie.

C’est un rappel constant de la mécanique.

C’est essentiel, car le module ne doit pas donner l’impression d’être un simple assemblage d’éléments “cool”.

Zone B — La scène à construire

L’élève a :
5 emplacements maximum,
8 jetons maximum.

Et il doit “acheter” des éléments.

Pourquoi ce système visuel de jetons ?

Parce qu’il matérialise de manière ludique la contrainte.

Un enfant comprend instinctivement :
que les ressources sont limitées,
que chaque choix coûte quelque chose,
que choisir, c’est renoncer.

Le système de jetons remplace une explication théorique sur :
la faisabilité,
la densité,
le coût narratif,
la surcharge.

C’est une grande force du module.

13. LA LOGIQUE DES JETONS — POURQUOI ELLE EST FONDAMENTALE

Les jetons symbolisent plusieurs choses à la fois :
la complexité,
le coût de production,
le poids d’un choix dans une scène.

On ne le dit pas techniquement aux élèves.

On ne dit pas :

ceci représente un surcoût de production.

On leur dit simplement, en substance :

chaque choix a un coût.

C’est beaucoup plus puissant.

14. POURQUOI CERTAINS ÉLÉMENTS COÛTENT 0, 1, 2 OU 3

Nous avons organisé les éléments en niveaux.

0 jeton

Exemples :
dialogue,
silence,
gros plan,
ralenti,
petit objet symbolique.

Pourquoi ?

Parce que ces éléments ne sont pas “spectaculaires”, mais ils peuvent être très puissants narrativement.

Le fait de les rendre peu coûteux transmet un message très fort :

tu n’as pas besoin d’un gros effet pour faire une scène forte.

1 jeton

Exemples :
mensonge,
secret caché,
révélation,
conflit verbal,
rivalité explicite.

Pourquoi ?

Parce que ces éléments ajoutent une vraie couche dramatique, mais restent accessibles.

Ils sont intéressants narrativement sans être trop lourds.

2 jetons

Exemples :
tournage de nuit,
musique importante,
personnage secondaire,
groupe / figurants,
déplacement / véhicule.

Pourquoi ?

Parce qu’ils alourdissent la scène, soit en production, soit en lisibilité.

Le but est que l’élève sente qu’ajouter des couches complique l’ensemble.

3 jetons

Exemples :
explosion,
accident majeur,
effet spécial fort,
cascade,
changement d’époque.

Pourquoi ?

Parce qu’on veut casser une illusion très fréquente chez les jeunes :

plus c’est spectaculaire, meilleure est la scène.

Non.
Plus c’est spectaculaire, plus cela demande d’être justifié.

15. POURQUOI LE MODULE N’EST PAS “ANTI-SPECTACULAIRE”

C’est important.

Le module ne dit pas :

le spectaculaire est nul.

Il dit :

le spectaculaire coûte cher.
Donc, si tu le choisis, il faut qu’il serve vraiment ton histoire.

Cette nuance est essentielle.

16. LE RÔLE DU FEEDBACK FINAL

Quand un élève valide sa scène, l’outil ne bloque pas.

Il donne un feedback.

Exemples :
ta scène est spectaculaire, mais on comprend moins ce que veut le personnage,
ton histoire est claire, comment la rendre plus forte visuellement,
on voit bien le conflit et le changement.

Pourquoi ce feedback est non bloquant ?

Parce qu’on ne veut pas humilier ni sanctionner.

Le but est que l’élève ressente une orientation, pas une note.

Le module est un terrain d’expérimentation.

17. LA PHASE COLLECTIVE FINALE — POURQUOI ELLE EST OBLIGATOIRE

Le jeu pourrait s’arrêter à la validation individuelle.

Mais ce serait une erreur.

Il faut une phase finale où l’on compare deux scènes contrastées.

Pourquoi ?

Parce que c’est le moment où les élèves voient clairement que :
certaines scènes sont plus lisibles que d’autres,
certaines scènes sont plus fortes sans être plus chargées,
ils ne sont pas tous d’accord sur ce qui fonctionne.

Et c’est précisément cette discussion qui fait monter la compréhension.

18. CE QUE L’INTERVENANT DOIT OBSERVER DANS LE MODULE 2

Ce module est extrêmement riche en indices.

L’intervenant doit observer :

les élèves qui empilent des éléments spectaculaires

Cela peut traduire :
une envie d’en mettre trop,
une difficulté à faire confiance au simple,
une confusion entre action et dramaturgie.

les élèves qui construisent des scènes très sobres mais claires

Cela peut révéler :
une compréhension instinctive de la structure,
une capacité à organiser.

les élèves qui comprennent immédiatement la logique “veut / mais / fin”

Ils seront souvent très utiles plus tard.

les élèves qui bloquent

Il faut repérer s’ils bloquent sur :
l’écriture,
l’imagination,
la compréhension de la scène.

19. LES PIÈGES PÉDAGOGIQUES DU MODULE 2

Le Module 2 peut échouer de plusieurs façons si l’intervenant le pilote mal.

piège 1 — Trop expliquer

Si l’intervenant théorise trop :
la tension du jeu tombe,
les élèves se mettent à chercher “la bonne réponse”.

piège 2 — Trop laisser faire

À l’inverse, si on ne recadre jamais :
certains élèves ne comprennent pas ce qu’ils apprennent.

piège 3 — Valoriser les scènes les plus “énormes”

Si l’intervenant se laisse séduire par le spectaculaire, il casse le message du module.

piège 4 — Faire croire qu’il y a une formule magique

Le module ne doit pas être perçu comme :

pour faire une bonne scène, il suffit de cocher des cases.

Il faut toujours rappeler :

ce sont des outils, pas une recette automatique.

20. CE QUE LE MODULE 2 PRÉPARE POUR LA SUITE

Le module 2 prépare directement le module 3.

Sans le module 2, le module 3 serait juste :

invente une idée.

Avec le module 2, le module 3 devient :

trouve une idée qui peut réellement devenir une histoire.

C’est une différence énorme.

21. INTERSÉANCE ENTRE MODULE 1 ET MODULE 2

Même si elle est légère, cette interséance a un rôle important.

Les données collectées au module 1 permettent de dégager :
des tendances d’observation,
des tendances de projection narrative,
des profils plus descriptifs ou plus interprétatifs.

Pourquoi c’est utile ?

Parce que le module 2 ne doit pas être lancé “à l’aveugle”.

L’intervenant ou l’outil peut déjà savoir :
si la classe est très littérale,
si elle est déjà imaginative,
si elle a besoin d’être davantage guidée.

22. INTERSÉANCE ENTRE MODULE 2 ET MODULE 3

Cette interséance est plus importante.

Les résultats du module 2 permettent de faire ressortir :
les émotions dominantes choisies,
les types de scènes les plus réussies,
les grands thèmes qui traversent la classe,
les premiers indices de compréhension narrative.

Pourquoi c’est important ?

Parce que le module 3 va demander aux élèves de produire un “Et si…”.

Plus on comprend déjà ce vers quoi la classe penche, plus on peut préparer des aides pertinentes.

Le module 3 n’est donc pas déconnecté du module 2.
Il en est la continuation directe.

23. POURQUOI LE MODULE 2 EST UN MODULE CLÉ DANS BANLIEUWOOD

Si le module 1 dit :

tu peux imaginer,

le module 2 dit :

mais pour qu’une scène tienne, il faut choisir.

C’est le premier module où l’élève découvre que la création n’est pas juste de la liberté.
C’est aussi de la contrainte.

Et c’est précisément pour ça que ce module est aussi important :
il installe la réalité de l’écriture sans tuer le plaisir de créer.

24. RÉSUMÉ STRATÉGIQUE DU MODULE 2

Si tu dois expliquer à ton associé en une phrase le rôle du module 2, ce serait :

Le module 2 fait passer les élèves de “j’invente quelque chose” à “je comprends qu’une scène fonctionne parce qu’elle repose sur une intention, un obstacle, un changement, et des choix sous contrainte”.
