# 05_MANUEL MODULE 

MANUEL DE FORMATION INTERVENANT MÉTHODE BANLIEUWOOD

1. L’OBJECTIF PROFOND DE BANLIEUWOOD

Banlieuwood n’est pas un atelier de cinéma.

Le cinéma est un prétexte pédagogique.

L’objectif réel est de faire découvrir aux élèves :
leur capacité à observer
leur capacité à imaginer
leur capacité à collaborer
leur capacité à structurer une idée.

Le film final est important, mais il n’est pas le cœur de la méthode.

Le cœur de la méthode est le chemin que les élèves parcourent pour y arriver.

3. LE RÔLE DE L’INTERVENANT

L’intervenant n’est pas un professeur.

Il est :
un guide
un animateur
un facilitateur.

Il doit éviter trois erreurs.

erreur 1

Corriger les élèves.

Dans Banlieuwood, on ne corrige pas une imagination.

erreur 2

Donner les réponses.

Le rôle est de poser les bonnes questions.

erreur 3

Accélérer les décisions.

Les discussions font partie de l’apprentissage.
