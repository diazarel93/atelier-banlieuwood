# MODULE 3

MODULE 3 — DOCUMENT DE CONCEPTION DÉTAILLÉ

Le “Et si…” — la naissance de l’idée

1. PLACE DU MODULE 3 DANS LA MÉTHODE

Le Module 3 arrive après le Module 2.

Le Module 1 a permis aux élèves de comprendre que :

chacun peut voir une image différemment.

Le Module 2 leur a permis de comprendre que :

une scène tient parce qu’elle repose sur une structure, des tensions, et des choix.

Le Module 3 arrive donc à un moment très précis :
les élèves ont déjà commencé à sentir ce qu’est une scène, mais ils n’ont pas encore créé de véritable point de départ narratif personnel.

Le Module 3 sert exactement à cela :

donner à chaque élève une première vraie graine d’histoire.

Autrement dit, le Module 3 est le moment où l’imaginaire cesse d’être seulement réactif pour devenir générateur.

2. OBJECTIF PROFOND DU MODULE 3

Le but du Module 3 n’est pas de faire écrire une histoire.

Le but du Module 3 est beaucoup plus précis :

amener chaque élève à produire au moins un “Et si…” suffisamment solide pour devenir une base de récit.

Le “Et si…” n’est pas :
un synopsis,
un scénario,
une structure,
ni même un vrai pitch.

C’est une étincelle orientée.

Le Module 3 doit donc réussir un équilibre délicat :
laisser de la liberté,
mais éviter le flou,
stimuler l’imaginaire,
mais sans partir dans tous les sens,
faire naître des idées,
sans exiger un niveau d’écriture trop élevé.

3. POURQUOI LE MODULE 3 EST INTROSPECTIF

Nous avons clairement validé que le Module 3 est un module introspectif.

Ce choix est fondamental.

Pourquoi ?

Parce que jusqu’ici, les élèves ont déjà vécu :
de l’observation collective,
des confrontations,
des discussions,
des votes,
des mises en commun.

Le Module 3, lui, doit créer un espace plus personnel.

À ce moment de la progression, l’élève doit se retrouver face à lui-même, à ses images mentales, à ce qu’il projette spontanément.

Le module doit donc créer une expérience où l’élève sent :

“cette idée peut venir de moi.”

C’est essentiel pour la suite, car le pitch du Module 4 ne peut être fort que si le point de départ a été un minimum approprié intérieurement.

4. POURQUOI LE MODULE 3 REPOSE SUR LE “ET SI”

Le “Et si…” a été choisi comme structure centrale pour plusieurs raisons.

Raison 1 — C’est universel

Le “Et si…” est une porte d’entrée très simple dans la fiction.

Même un enfant qui ne connaît rien à la structure peut comprendre :
Et si un élève disparaissait…
Et si un téléphone révélait tous les secrets…
Et si un prof mentait…

Raison 2 — C’est court

On ne demande pas encore une histoire complète.
On ne demande qu’une hypothèse de départ.

Cela réduit la peur de mal faire.

Raison 3 — C’est dynamique

Le “Et si…” contient déjà une tension.
Il ouvre immédiatement vers une possibilité.

Raison 4 — C’est compatible avec tout le reste

Le “Et si…” peut ensuite être transformé :
en pitch,
en structure,
en scénario,
en film collectif.

Donc ce n’est pas juste une astuce pédagogique.
C’est un vrai format de base narratif.

5. POURQUOI NOUS N’AVONS PAS DEMANDÉ AUX ÉLÈVES “D’INVENTER UNE HISTOIRE”

Nous avons volontairement évité de dire :

“Invente une histoire.”

Pourquoi ?

Parce que cette consigne est trop grande, trop intimidante, trop floue.

Chez beaucoup d’élèves, elle déclenche immédiatement :
le vide,
la peur de mal faire,
ou des réponses clichés.

Le “Et si…” est beaucoup plus accessible, car il ne demande pas :
de tout construire,
de tout prévoir,
de tout réussir.

Il demande seulement :

de trouver une bonne porte d’entrée.

Et c’est tout l’enjeu du Module 3.

6. LA STRUCTURE GLOBALE DU MODULE 3

Le module repose sur quatre grands temps :
une introduction collective courte, qui cadre l’imaginaire
une exploration d’images
un espace de travail individuel centré sur un seul “Et si…”
une clôture qui prépare le passage au pitch.

Le module doit durer environ 35 à 40 minutes.

C’est important, car il doit rester tendu et vivant.
S’il devient trop long, l’introspection se transforme en fatigue.

7. INTRODUCTION COLLECTIVE — POURQUOI ELLE EXISTE

Le module commence par une introduction orale de l’intervenant.

Mais cette introduction ne doit pas ressembler à un cours.

Elle a trois objectifs.

1. Faire comprendre ce qu’est un “Et si…”

L’intervenant donne 3 ou 4 exemples connus.

Exemple :
Harry Potter : et si un enfant ordinaire apprenait qu’il est sorcier ?
Vice-Versa : et si les émotions d’une petite fille étaient des personnages ?
etc.

2. Vérifier que la classe a compris

Les élèves lèvent la main s’ils reconnaissent au moins un exemple.

Pourquoi ?
Parce qu’il faut être sûr qu’ils aient saisi la logique du “Et si…”.

3. Cadre de l’atelier

L’intervenant précise que le terrain de jeu sera proche de l’univers scolaire.

Pourquoi ?
Parce que si on laisse les élèves dans une liberté totale à ce stade, on ouvre trop de portes :
violence,
clichés,
hors sujet,
univers très éloignés du réel scolaire.

Le cadre scolaire joue ici un rôle d’ancrage.

8. POURQUOI LE CADRE SCOLAIRE A ÉTÉ VALIDÉ

Nous avons clairement validé que le module 3 devait se dérouler dans un univers proche du collège.

Ce choix n’est pas une limitation arbitraire.

Il répond à une logique très forte :

1. C’est leur matière première la plus connue

Les élèves passent une grande partie de leur vie à l’école.

2. Cela augmente la probabilité d’avoir des idées

Même ceux qui pensent ne pas être imaginatifs peuvent projeter quelque chose dans un cadre qu’ils connaissent.

3. Cela réduit les idées ingérables

Si on les laisse partir dans tous les univers possibles, une grande partie des propositions sera inutilisable ou devra être censurée.

Or le but n’est pas de dire non à leurs idées.
Le but est de les amener dès le départ vers des idées exploitables.

4. Cela prépare la faisabilité

Le projet final devra pouvoir être tourné dans un cadre scolaire ou proche.

Donc ce choix est à la fois :
pédagogique,
narratif,
logistique.

9. LES IMAGES — POURQUOI LE MODULE 3 PASSE PAR DES IMAGES

Le cœur visuel du Module 3 repose sur un ensemble d’images.

Ce choix est central.

Pourquoi les images ?

Parce qu’elles servent de déclencheur d’imaginaire.

Un enfant qui bloque sur une page blanche peut très bien réagir à une image.

L’image :
rassure,
stimule,
suggère,
sans imposer un récit.

Elle permet donc de contourner le blocage de l’écriture pure.

10. POURQUOI LES IMAGES DOIVENT ÊTRE “LÉGÈREMENT DÉSÉQUILIBRÉES”

Les images du Module 3 ne doivent pas être neutres au sens plat.

Elles doivent être légèrement déséquilibrées.

C’est-à-dire :
une situation visiblement ordinaire,
mais avec quelque chose qui dérange légèrement.

Exemples validés :
une chaise vide dans une classe pleine,
un mot étrange écrit au tableau,
un élève seul dans un couloir,
un téléphone autour duquel plusieurs élèves rient,
un élève devant le bureau du proviseur,
un sac abandonné,
etc.

Pourquoi ce principe est très fort ?

Parce qu’il ouvre naturellement le “Et si…”.

L’élève n’a pas besoin d’inventer un univers entier.
Le déséquilibre de l’image appelle déjà une hypothèse.

11. POURQUOI NOUS AVONS VOULU MÉLANGER DES TONS

Nous avons validé qu’il fallait un panel d’images mélangeant :
du sérieux,
du léger,
du tendu,
du gênant,
du drôle,
du mystérieux.

Pourquoi ?

Parce que si toutes les images vont dans un seul ton :
les idées produites vont se ressembler,
certains profils d’élèves ne trouveront pas leur porte d’entrée.

Le mélange des tonalités permet :
à un élève plus dramatique d’entrer,
à un élève plus comique d’entrer,
à un élève plus étrange ou mystérieux d’entrer.

C’est essentiel pour la variété des idées futures.

12. POURQUOI 10 IMAGES

Nous avons validé un ensemble de 10 images.

Ce nombre est important.
Moins de 10 = risque de pauvreté de choix.
Beaucoup plus = surcharge cognitive.

Dix images, c’est suffisamment :
riche,
varié,
stimulant,

sans devenir fatigant.

13. CHOIX VISUEL DE L’EXPLORATION DES IMAGES

L’interface des images doit être conçue comme un espace de défilement ou de sélection fluide.

Pas comme un diaporama rigide.

L’élève doit sentir qu’il “parcourt un terrain d’idées”.

Pourquoi ?

Parce qu’on veut qu’il explore, pas qu’il subisse une suite d’images.

Le geste doit être simple :
regarder,
swiper,
sélectionner.

Et l’interface doit permettre trois niveaux d’engagement :
regarder sans cliquer,
cliquer “ça me parle”,
finalement choisir une image à travailler.

14. POURQUOI L’ÉLÈVE PEUT EN SÉLECTIONNER JUSQU’À 3

Nous avons validé qu’il pouvait marquer jusqu’à trois images.

Pourquoi pas une seule directement ?

Parce qu’on veut lui laisser :
un petit espace d’exploration,
la possibilité d’hésiter,
la possibilité de comparer ses élans.

Pourquoi pas plus de trois ?

Parce que sinon :
il sélectionne tout,
il ne hiérarchise rien,
et l’idée du choix disparaît.

Le maximum de trois images crée déjà une première contrainte légère.

15. POURQUOI IL DOIT ENSUITE N’EN CHOISIR QU’UNE

Une fois qu’il a exploré, il doit choisir une seule image à travailler.

Ce point est très important.

Le module ne doit pas produire :
dix débuts d’idées,
ou plusieurs “Et si…” flous.

Il doit produire :
au moins un “Et si…” clair,
approprié,
envoyé.

Le recentrage sur une seule image crée donc un premier acte de décision.

16. L’ESPACE DE TRAVAIL UNIQUE — POURQUOI TOUT EST SUR LE MÊME ÉCRAN

Une fois l’image choisie, nous avons validé un choix d’interface fondamental :
tout doit être réuni sur un seul espace de travail.

Cet écran contient :
l’image,
le champ du “Et si…”,
les QCM.

Pourquoi ce choix est très important ?

Parce qu’on veut éviter une logique de tunnel :
écran 1,
puis écran 2,
puis écran 3.

Ce serait trop scolaire, trop séquencé, trop cassant.

Le fait d’avoir un espace de travail unique permet :
de sentir l’ensemble,
de modifier à tout moment,
de faire des allers-retours mentaux,
de ne pas vivre le module comme une série de cases à cocher.

17. LA SEULE ÉCRITURE OBLIGATOIRE — POURQUOI CETTE DÉCISION EST CENTRALE

Nous avons validé une règle forte :

la seule écriture obligatoire du module est la phrase “Et si…”.

Tout le reste se fait en :
clics,
sélections,
QCM.

Pourquoi ce choix est crucial ?

Parce qu’il évite que le module devienne un exercice de rédaction déguisé.

On veut capter :
l’idée,
la direction,
l’élan narratif,

sans bloquer les élèves sur :
l’orthographe,
la formulation,
la peur de mal écrire.

C’est une décision clé de l’ADN Banlieuwood :
on privilégie l’accès à la pensée plutôt que la performance scolaire.

18. LES QCM — POURQUOI ILS EXISTENT ET À QUOI ILS SERVENT

Les QCM du Module 3 ne sont pas là pour “faire joli”.

Ils jouent plusieurs rôles.

1. Structurer l’idée sans forcer l’écriture

L’élève précise son idée par clics.

2. Maintenir le sentiment de jeu

Le clic est plus léger que la rédaction.

3. Récolter une data claire

Les réponses deviennent exploitables entre les séances.

4. Pousser l’élève à préciser son intuition

Par exemple :
ton,
personnage,
élément déclencheur,
type de fin.

L’élève ne construit pas encore une structure complète, mais il commence à orienter son idée.

19. LES QUESTIONS VALIDÉES DU QCM — LEUR LOGIQUE

Les questions retenues ont été pensées pour être :
simples,
évocatrices,
compréhensibles à cet âge,
et narrativement utiles.

On ne veut pas :
des questions de structure experte,
ni des questions trop psychologiques.

On veut des questions qui aident à faire exister l’idée.

Exemples validés :
ton de l’histoire,
type de personnage,
ce qui change tout,
durée de l’histoire,
type de fin,
motivation éventuelle du personnage.

Le rôle de ces questions est de transformer une intuition floue en hypothèse un peu plus dense.

20. POURQUOI LE MODULE RESTE TRÈS MODIFIABLE JUSQU’À L’ENVOI

Nous avons validé que l’élève puisse modifier :
son “Et si…”,
ses réponses,
ses choix,

jusqu’au moment où il clique sur :

“J’envoie mon idée”.

Pourquoi ?

Parce qu’on ne veut pas figer trop tôt.

Le Module 3 est un espace d’essai.

C’est important psychologiquement, car cela crée :
de la sécurité,
de la liberté,
de la réécriture légère,
un rapport vivant à l’idée.

21. POURQUOI IL Y A UN BOUTON “JE PARTAGE POUR AIDER”

Nous avons validé une mécanique très importante :
l’élève peut décider de partager son idée pour aider les autres.

Pourquoi ce bouton existe ?

Parce qu’on a pris en compte un phénomène réel :
dans une classe, certains élèves vont avoir plusieurs idées, et d’autres vont bloquer.

Or on ne veut pas que ceux qui ont plein d’idées :
s’ennuient,
ou gardent tout pour eux.

Le bouton “Je partage pour aider” transforme leurs idées supplémentaires en ressource collective.

C’est une très belle mécanique de solidarité.

22. LE MODE AIDE ACTIVABLE PAR L’INTERVENANT

L’aide ne doit pas être automatique.

Nous avons validé qu’elle soit activée par l’intervenant.

Pourquoi ?

Parce qu’on ne veut pas que les élèves se jettent tous immédiatement sur les idées des autres ou sur des aides générées.

L’objectif du Module 3 reste :

faire naître une vraie première étincelle personnelle.

L’aide doit donc intervenir seulement :
si ça bloque,
si le groupe en a besoin,
ou si l’intervenant sent qu’il faut relancer.

23. L’ATMOSPHÈRE PÉDAGOGIQUE DU MODULE 3

C’est un point très important.

Le Module 3 doit être vécu comme une séance :
calme,
concentrée,
introspective,
peu bavarde.

Nous avons clairement validé que ce n’est pas ici qu’il faut :
multiplier les discussions,
ou faire débattre la classe.

Le module doit laisser de l’espace intérieur.

Pourquoi ?

Parce que l’idée a besoin de silence pour émerger.

24. CE QUE L’INTERVENANT DOIT FAIRE ET NE PAS FAIRE

Ce qu’il doit faire
cadrer,
rassurer,
rappeler qu’on ne cherche pas une histoire entière,
encourager ceux qui bloquent,
activer l’aide si nécessaire.

Ce qu’il ne doit pas faire
commenter trop tôt les idées,
demander trop de développement,
imposer une direction trop vite,
transformer la séance en brainstorming collectif.

25. LA FIN DE SÉANCE — POURQUOI ELLE EST IMPORTANTE

Le module se clôt par un message très important.

Nous avons validé la métaphore :

une idée = une étincelle
une histoire = quand l’étincelle devient feu.

Pourquoi cette métaphore fonctionne bien ?

Parce qu’elle est :
visuelle,
simple,
mémorisable,
et juste.

Elle dit exactement ce qu’on veut faire comprendre :
le “Et si…” n’est pas encore une histoire complète.
C’est seulement un début, une énergie, une promesse.

26. CE QUE LE MODULE 3 DOIT PRODUIRE CONCRÈTEMENT

À la fin du Module 3, chaque élève doit idéalement avoir :
au moins un “Et si…” envoyé,
un ton,
une direction de personnage,
une première coloration narrative.

Même si l’idée est simple.
Même si elle est imparfaite.
Même si elle est cliché.

Pourquoi ?
Parce qu’on sait déjà que le Module 4 et les interséances permettront de retravailler, challenger et clarifier.

Le Module 3 ne cherche pas la qualité finale.
Il cherche l’émergence.

27. INTERSÉANCE MODULE 3 → MODULE 4

Cette interséance est capitale.

Elle ne se voit pas côté élève, mais elle prépare tout le travail du module suivant.

Ce qui est produit :
un pitch miroir imparfait, basé uniquement sur le “Et si…” et les QCM.

Pourquoi un pitch miroir imparfait ?

Parce qu’il doit refléter ce que l’élève a réellement envoyé, même si c’est bancal.

L’idée n’est pas encore de corriger.
L’idée est de matérialiser.

Cela permettra ensuite au module suivant de mettre l’élève face à sa propre matière et de la travailler.

28. CE QUE L’INTERSÉANCE COMMENCE À PRÉPARER EN SILENCE

En plus du pitch miroir, cette interséance commence déjà à accumuler de quoi produire plus tard :
des pitchs directionnels plus cohérents,
des regroupements d’idées,
une lecture plus fine des envies narratives,
une base pour les futurs groupes.

Le Module 3 est donc à la fois :
un module de naissance d’idée,
et un module de récolte de matière.

29. LES PIÈGES DU MODULE 3

Le module peut échouer si on tombe dans certains travers.

piège 1 — Le rendre trop scolaire

Si on demande trop d’écriture, on perd la moitié de la classe.

piège 2 — Le rendre trop libre

Si on ne cadre pas assez le champ des possibles, on obtient beaucoup d’idées ingérables.

piège 3 — Intervenir trop fort

Si l’intervenant oriente trop vite les idées, les élèves n’ont plus l’impression que cela vient d’eux.

piège 4 — Vouloir déjà de la structure

Le Module 3 n’est pas là pour construire une histoire.
Il est là pour produire une bonne étincelle.

30. RÉSUMÉ STRATÉGIQUE DU MODULE 3

Si tu dois le résumer en une phrase à ton associé :

Le Module 3 transforme l’imaginaire libre des élèves en véritables graines d’histoire grâce à un système d’images, de “Et si…”, et de cadrage léger qui stimule sans enfermer.

Et si tu veux résumer sa fonction dans la méthode :

C’est le module où l’élève cesse seulement de réagir à des images et commence à produire un point de départ narratif personnel, exploitable pour toute la suite.
