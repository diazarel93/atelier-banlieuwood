# MODULE 1

 MODULE 1

LE REGARD

OBJECTIF RÉEL DU MODULE

Le module 1 a un objectif très précis.
Faire comprendre aux élèves que : leur regard est unique.

C’est une étape fondamentale. Beaucoup d’élèves pensent qu’ils ne sont pas créatifs.
Le module 1 sert à casser cette croyance.

STRUCTURE DU MODULE

Le module est construit autour de trois expériences.
imaginer
observer
interpréter

PARTIE 1

LES QUESTIONS DE POSITIONNEMENT

Durée : environ 5 minutes.

Les élèves répondent à quelques questions simples sur tablette.

Exemples :
regardes-tu des films ou des séries ?
préfères-tu jouer un personnage ou inventer une histoire ?
as-tu déjà imaginé une scène dans ta tête ?

POURQUOI CETTE ÉTAPE EXISTE

Elle a plusieurs fonctions.

fonction 1

Créer un engagement immédiat. Les élèves participent dès le début.

fonction 2

Donner à l’intervenant une lecture rapide de la classe.

Certains groupes sont très cinéphiles. D’autres pas du tout.

fonction 3

Faire comprendre aux élèves que leurs réponses comptent.

CE QUE L’INTERVENANT DOIT OBSERVER
les élèves qui répondent très vite
ceux qui hésitent
ceux qui semblent déjà très imaginatifs.

Ces indices seront utiles plus tard.

PARTIE 2

IMAGE 1 : PROJECTION NARRATIVE

Une image neutre apparaît.

Exemple :

Un élève seul dans un couloir.

Consigne :

Imagine ce qu’il se passe juste après cette image.

Les élèves écrivent.

POURQUOI CET EXERCICE EST ESSENTIEL

Il permet de révéler plusieurs choses.

1

Certains élèves écrivent une action simple.

Exemple :

Il rentre chez lui.

2

Certains inventent déjà une histoire.

Exemple :

Il vient d’être accusé à tort.

3

Certains inventent un univers entier.

Ces différences ne sont pas jugées.

Elles sont observées.

CE QUE L’INTERVENANT DOIT FAIRE

Ne pas commenter les réponses pendant l’écriture.

Laisser les élèves imaginer librement.

PARTIE 3

CONFRONTATION DES HISTOIRES

L’intervenant choisit deux réponses très différentes.

Il les lit anonymement.

CE QUI SE PASSE DANS LA CLASSE

Les élèves découvrent que :
deux histoires complètement différentes
peuvent naître de la même image.

LE MESSAGE PÉDAGOGIQUE

C’est une révélation pour beaucoup d’élèves.

Ils comprennent que :

l’imagination n’a pas de bonne ou mauvaise réponse.

PARTIE 4

IMAGE 2 : OBSERVATION

Une nouvelle image apparaît.

Cette fois la consigne change.

Les élèves doivent :
décrire ce qu’ils voient
expliquer ce que ces détails peuvent signifier.

POURQUOI CETTE ÉTAPE EST IMPORTANTE

Les élèves apprennent la différence entre :

observer et interpréter.

Beaucoup mélangent les deux.

EXEMPLE

Observation :

un sac est posé sur une chaise.

Interprétation :

quelqu’un est parti en courant.

PARTIE 5

IMAGE 3 : AMBIGUÏTÉ

Une image plus ambiguë est présentée.

Exemple :

un élève sourit devant un tableau.

Son nom est écrit dessus.

Punition ou récompense ?

OBJECTIF

Faire comprendre que :

les images peuvent être interprétées de plusieurs façons.

PARTIE 6

LE CARNET D’IDÉES

Les élèves reçoivent une mission simple.

Observer une situation réelle dans leur vie.

Exemples :
une dispute
un moment gênant
un moment drôle
une injustice.

POURQUOI CETTE MISSION EST IMPORTANTE

Elle apprend aux élèves que :

les histoires viennent souvent de la réalité.

CE QUE L’INTERVENANT DOIT DIRE

Les meilleurs films racontent souvent des choses qui existent vraiment.

RÉSULTAT DU MODULE 1

À la fin du module 1 :

les élèves comprennent que :
ils peuvent imaginer
ils peuvent observer
leurs idées ont de la valeur.

TRANSITION VERS LE MODULE 2

L’intervenant explique :

Inventer une histoire est une chose.
Mais comprendre comment fonctionne une scène en est une autre.
