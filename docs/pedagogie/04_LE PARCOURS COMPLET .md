# 04_LE PARCOURS COMPLET 

LE PARCOURS COMPLET D’UN ÉLÈVE DANS LE PROGRAMME BANLIEUWOOD

1. Objectif du document

Ce document décrit l’expérience vécue par un élève participant au programme Banlieuwood.

Il permet de comprendre :

comment les élèves découvrent le dispositif
comment ils participent aux différentes activités
comment une idée se transforme progressivement en film.

Ce document sert à illustrer concrètement la progression pédagogique du programme.

2. L’arrivée dans l’atelier

Lors de la première séance, les élèves découvrent le dispositif Banlieuwood.

Chaque élève dispose d’une tablette qui lui permettra de participer aux différentes activités.

L’intervenant explique que les élèves vont vivre une expérience particulière :

créer collectivement l’histoire d’un film.

Les élèves comprennent rapidement qu’ils ne vont pas seulement regarder ou analyser des films.

Ils vont participer activement à leur création.

3. Les premières activités : apprendre à regarder

Le programme commence par une étape essentielle : apprendre à observer.

Les élèves découvrent plusieurs images.

On leur demande d’imaginer ce qui se passe dans ces images ou ce qui pourrait se produire ensuite.

Rapidement, les élèves réalisent que leurs interprétations sont différentes.

Une même image peut susciter des histoires complètement différentes.

Cette première étape permet de développer le regard et l’imagination.

4. Comprendre comment fonctionne une scène

Après cette première exploration, les élèves découvrent que les histoires ne sont pas seulement une succession d’événements.

Ils apprennent qu’une scène repose sur quelques éléments fondamentaux :

un personnage
un objectif
un obstacle
un changement.

À travers des exercices ludiques, les élèves comprennent que les histoires naissent souvent de tensions ou de conflits.

Ils commencent à percevoir les mécanismes qui structurent une narration.

5. Inventer une idée d’histoire

Dans les modules suivants, les élèves sont invités à imaginer leurs propres idées d’histoires.

Pour cela, ils utilisent la méthode du “Et si…”.

Par exemple :

Et si un élève découvrait un secret caché dans son école ?

Et si un objet étrange apparaissait dans la cour ?

Les élèves comprennent que les histoires commencent souvent par une hypothèse simple.

Cette étape libère l’imagination et encourage la créativité.

6. Transformer une idée en pitch

Une fois les idées générées, les élèves apprennent à structurer leurs propositions.

Ils doivent préciser :

qui est le personnage principal
ce que ce personnage veut
ce qui l’empêche d’obtenir ce qu’il veut.

Les élèves découvrent ensuite la notion de pitch.

Un pitch est une manière courte et claire de présenter une idée d’histoire.

Chaque élève est invité à formuler son pitch en quelques phrases.

7. Construire une histoire collective

À ce stade du programme, les idées individuelles commencent à se transformer en projet collectif.

Les élèves votent pour certaines propositions.

Les éléments les plus intéressants sont sélectionnés.

Peu à peu, les élèves construisent ensemble les éléments principaux de l’histoire :

le ton du film
la situation de départ
les personnages
le problème central.

Les élèves comprennent que la création d’un film implique souvent des discussions et des choix collectifs.

8. Améliorer le scénario

Une fois l’histoire définie, les élèves travaillent à partir d’un scénario de base.

Ce scénario contient volontairement des zones à compléter.

Les élèves doivent :

améliorer certains dialogues
clarifier certaines situations
renforcer les moments importants.

Ils travaillent en petits groupes et proposent leurs idées à la classe.

Les décisions finales sont prises collectivement.

Cette étape leur permet de découvrir que l’écriture d’un film est un travail progressif.

9. Découvrir la grammaire du cinéma

Dans les modules suivants, les élèves apprennent comment une scène peut être filmée.

Ils découvrent les bases de la grammaire visuelle du cinéma :

plan large
plan moyen
gros plan
plan de réaction.

Ils comprennent que le choix d’un plan influence la manière dont le spectateur perçoit une scène.

Les élèves réalisent ensuite un mini découpage pour certaines scènes de leur propre film.

10. Former l’équipe du film

À l’approche du tournage, les élèves découvrent les différents rôles présents sur un plateau de cinéma.

Par exemple :

réalisateur
cadreur
ingénieur son
assistant réalisateur
script
acteur.

Les élèves choisissent ou se voient proposer des rôles en fonction de leurs affinités et de leur participation dans les modules précédents.

Chaque élève devient ainsi responsable d’une partie du projet.

11. Préparer le tournage

Une fois les rôles répartis, les élèves préparent le tournage.

Ils identifient :

les décors
les accessoires
les scènes à filmer.

Ils utilisent également un mini découpage pour organiser les plans nécessaires.

Les élèves découvrent ainsi que la préparation est une étape essentielle dans la réalisation d’un film.

12. Une expérience collective

Tout au long du programme, les élèves participent à une aventure collective.

Ils découvrent que la création artistique peut être :

un espace d’expression
un lieu de coopération
une expérience partagée.

Même si tous les élèves ne deviennent pas cinéastes, ils repartent avec une compréhension nouvelle du processus de création.

Conclusion

Le parcours Banlieuwood permet aux élèves de passer progressivement :

de l’observation à la création
de l’idée individuelle à l’histoire collective
de l’imagination à la préparation d’un film.

Cette progression leur permet de comprendre concrètement comment naît une œuvre audiovisuelle.

L’objectif principal n’est pas seulement de produire un film, mais de faire vivre aux élèves une expérience complète de création.
