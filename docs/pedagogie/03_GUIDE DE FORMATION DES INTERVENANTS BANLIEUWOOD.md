# 03_GUIDE DE FORMATION DES INTERVENANTS BANLIEUWOOD

GUIDE DE FORMATION DES INTERVENANTS BANLIEUWOOD

1. Objectif du document

Ce document a pour objectif de former les intervenants qui animeront les ateliers Banlieuwood.

Il explique :

la posture attendue
les principes pédagogiques
la manière d’animer les modules
les erreurs à éviter.

Ce guide permet de garantir que l’expérience Banlieuwood reste cohérente quel que soit l’intervenant.

2. Le rôle de l’intervenant

L’intervenant Banlieuwood n’est pas un professeur au sens classique.

Son rôle n’est pas d’évaluer les élèves ni de leur donner des notes.

L’intervenant agit comme :

un facilitateur
un guide
un animateur de création collective.

Il accompagne les élèves dans le processus de création.

Il aide à structurer les idées, mais il ne décide pas à leur place.

3. La posture de l’intervenant

L’intervenant doit adopter une posture particulière.

Il doit être :

bienveillant
encourageant
curieux des idées des élèves.

L’objectif est de créer un espace où les élèves se sentent libres de proposer des idées.

Les élèves ne doivent jamais avoir peur de se tromper.

4. L’importance de l’écoute

Dans les ateliers Banlieuwood, les idées viennent souvent des élèves.

L’intervenant doit être attentif aux propositions du groupe.

Il doit savoir :

écouter
reformuler
valoriser certaines idées.

Même une idée maladroite peut devenir intéressante si elle est retravaillée.

5. Éviter les jugements

L’intervenant doit éviter les jugements directs.

Par exemple, il ne doit pas dire :

“Cette idée n’est pas bonne.”

Il peut plutôt dire :

“Comment pourrait-on rendre cette idée plus claire ?”

L’objectif est de transformer les critiques en questions.

6. Encourager la participation

Certains élèves participent spontanément.

D’autres peuvent rester en retrait.

L’intervenant doit veiller à encourager la participation de tous.

L’utilisation des tablettes permet déjà à chaque élève de s’exprimer.

Mais l’intervenant peut également solliciter des élèves plus discrets en leur posant des questions simples.

7. Maintenir un rythme dynamique

Les ateliers Banlieuwood doivent rester dynamiques.

L’intervenant doit éviter les temps morts.

Pour cela, il doit :

enchaîner les activités
relancer les discussions
garder un rythme vivant.

Un atelier trop lent peut entraîner une perte d’attention.

8. Gérer les discussions collectives

Les discussions collectives sont un moment important des ateliers.

L’intervenant doit aider les élèves à structurer leurs échanges.

Il peut par exemple poser des questions simples :

Qui est le personnage principal ?
Que veut-il ?
Qu’est-ce qui l’empêche de réussir ?

Ces questions permettent de guider la réflexion sans imposer une réponse.

9. Utiliser l’outil numérique

L’outil numérique Banlieuwood est conçu pour accompagner les ateliers.

Il permet notamment de :

collecter les réponses des élèves
organiser les votes
structurer les activités.

L’intervenant doit utiliser cet outil comme un support.

Cependant, il ne doit pas laisser la tablette remplacer les échanges humains.

10. Valoriser les idées des élèves

Un principe essentiel de Banlieuwood est que les élèves doivent reconnaître leurs idées dans le projet final.

L’intervenant doit donc valoriser les contributions individuelles.

Même si certaines idées ne sont pas retenues, elles peuvent inspirer d’autres propositions.

11. Gérer les frustrations

Dans un projet collectif, certaines idées ne seront pas retenues.

L’intervenant doit expliquer aux élèves que cela fait partie du processus créatif.

Il peut rappeler que :

les films sont souvent le résultat de nombreux choix collectifs.

Cette approche permet de transformer la frustration en compréhension du travail d’équipe.

12. Maintenir un climat bienveillant

L’intervenant doit veiller à maintenir un climat respectueux dans le groupe.

Les élèves doivent pouvoir exprimer leurs idées sans être moqués.

Toute forme de moquerie ou de dévalorisation doit être immédiatement recadrée.

Le respect est une condition essentielle pour que la créativité puisse s’exprimer.

13. Adapter l’atelier au groupe

Chaque classe est différente.

L’intervenant doit être capable d’adapter légèrement l’animation en fonction du groupe.

Par exemple :

si le groupe est très dynamique, il peut encourager les débats
si le groupe est plus réservé, il peut poser des questions simples pour relancer la participation.

14. L’objectif final

L’objectif principal de Banlieuwood n’est pas seulement de produire un film.

Le film est un prétexte pour permettre aux élèves de vivre une expérience de création.

Les élèves doivent repartir avec :

une meilleure compréhension de la création artistique
une expérience collective positive
la sensation d’avoir participé à quelque chose.

Conclusion

Le rôle de l’intervenant est essentiel dans la réussite du programme Banlieuwood.

L’outil numérique structure l’expérience, mais l’énergie de l’atelier dépend de l’intervenant.

Un bon intervenant Banlieuwood doit :

écouter
encourager
structurer
et accompagner la création collective.

C’est cette posture qui permet aux élèves de vivre pleinement l’expérience.
