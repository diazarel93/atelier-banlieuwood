# 06_SCHÉMA GLOBAL DU PROGRAMME BANLIEUWOOD

SCHÉMA GLOBAL DU PROGRAMME BANLIEUWOOD

1. Objectif du document

Ce document présente la structure globale du programme Banlieuwood.

Il permet de comprendre :
la logique pédagogique du projet
la progression des élèves
la transformation progressive d’une idée en film
la place de chaque module dans cette progression

Ce document sert de référence synthétique pour l’ensemble des partenaires, intervenants et développeurs impliqués dans le projet.

2. Contexte du programme

Banlieuwood est un dispositif pédagogique qui utilise la création d’un film comme outil d’apprentissage artistique et créatif.

L’objectif n’est pas seulement de faire découvrir le cinéma, mais de permettre aux élèves de :

développer leur imagination
comprendre les mécanismes narratifs
expérimenter le travail collectif
découvrir les métiers du cinéma
transformer une idée en projet concret.

Le programme repose sur une progression en huit modules.

Chaque module correspond à une étape dans la transformation d’une simple observation vers la réalisation d’un film collectif.

3. La logique pédagogique globale

Le programme Banlieuwood suit une progression structurée.

Chaque module correspond à une compétence spécifique.

Module 1 : Observer
Les élèves apprennent à regarder une image et à comprendre qu’elle peut être interprétée de différentes façons.

Module 2 : Comprendre
Les élèves découvrent les mécanismes qui composent une scène : personnage, intention, obstacle et tension.

Module 3 : Imaginer
Les élèves apprennent à générer une idée d’histoire grâce à la méthode du “Et si…”.

Module 4 : Clarifier
Les élèves transforment leur idée en pitch en définissant un personnage, son objectif et l’obstacle qu’il rencontre.

Module 5 : Construire
Les élèves assemblent leurs idées pour créer une histoire collective.

Module 6 : Écrire
Les élèves transforment l’histoire collective en scénario grâce à un scénario de base qu’ils complètent et améliorent.

Module 7 : Visualiser
Les élèves découvrent la grammaire visuelle du cinéma et apprennent à transformer une scène en images.

Module 8 : Produire
Les élèves forment une équipe de tournage et préparent concrètement la réalisation du film.

4. Le parcours pédagogique détaillé

Module 1 : Observer

L’objectif du premier module est d’apprendre aux élèves à regarder une image.

Les élèves découvrent que chacun peut percevoir une image différemment.

Ils comprennent que l’interprétation dépend du regard de chacun.

Ce module développe :

l’observation
l’interprétation
le regard critique.

Module 2 : Comprendre

Les élèves apprennent à comprendre ce qui se joue dans une scène.

Ils découvrent que toute situation narrative repose sur plusieurs éléments essentiels :

un personnage
une intention
un obstacle
une tension.

Ce module développe :

la lecture narrative
la compréhension dramatique.

Module 3 : Imaginer

Les élèves apprennent à générer des idées d’histoires.

Ils utilisent la méthode du “Et si…” pour créer des hypothèses narratives.

Par exemple :

Et si un élève découvrait un secret caché dans l’école ?

Ce module développe :

l’imagination
la créativité
la capacité à formuler une idée narrative.

Module 4 : Clarifier

Les élèves apprennent à transformer une idée en pitch.

Ils précisent :

qui est le personnage
ce que le personnage veut
ce qui l’empêche d’atteindre son objectif.

Ils apprennent également à raconter leur idée en moins de trente secondes.

Ce module développe :

la synthèse
la clarté narrative
la capacité à structurer une idée.

Module 5 : Construire

Les idées individuelles deviennent des éléments d’une histoire collective.

Les élèves construisent l’histoire en choisissant progressivement :

le ton
la situation
les personnages
l’objectif
l’obstacle.

Ce module développe :

la coopération
la construction collective
la prise de décision.

Module 6 : Écrire

Les élèves travaillent à partir d’un scénario de base.

Ce scénario contient volontairement des zones à compléter.

Les élèves améliorent le scénario en :

précisant les scènes
ajoutant des dialogues
renforçant les tensions
corrigeant les incohérences.

Ce module développe :

l’écriture collaborative
l’analyse narrative
la construction d’un récit.

Module 7 : Visualiser

Les élèves découvrent comment une scène devient une image.

Ils apprennent les bases de la mise en scène :

plan large
plan moyen
gros plan
plan réaction.

Ils réalisent un mini découpage des scènes clés.

Ce module développe :

la grammaire visuelle du cinéma
la préparation du tournage
la compréhension de la mise en scène.

Module 8 : Produire

Les élèves découvrent les métiers du cinéma et forment une équipe de tournage.

Les rôles peuvent inclure :

réalisateur
cadreur
ingénieur son
assistant réalisateur
script
acteurs.

Chaque élève reçoit un document simple correspondant à son rôle.

Ce module développe :

le travail d’équipe
le sens des responsabilités
l’organisation d’un tournage.

5. La logique globale de transformation

Le programme Banlieuwood suit une transformation progressive.

Les élèves passent par plusieurs étapes :

Regarder
Analyser
Inventer
Structurer
Coopérer
Écrire
Mettre en scène
Réaliser.

Cette progression reproduit, de manière simplifiée, les étapes réelles de fabrication d’un film.

6. Le rôle de l’outil numérique

L’outil numérique Banlieuwood accompagne chaque module.

Il permet :

d’afficher les images
de guider les exercices
de collecter les idées
de structurer les choix collectifs
de générer certains contenus entre les séances.

L’outil ne remplace pas l’intervenant.

Il agit comme un support pédagogique interactif qui facilite l’expérience collective.

7. Les interséances

Entre certains modules, l’outil réalise des traitements automatiques.

Ces interséances permettent notamment de :

générer un scénario de base
créer des cartes narratives
produire des images de storyboard
préparer les documents nécessaires au tournage.

Ces traitements permettent d’éviter les temps morts pendant les ateliers et d’optimiser le temps pédagogique.

8. Résultat final du programme

À la fin du parcours, les élèves disposent :

d’un film écrit collectivement
d’un mini storyboard
d’une équipe de tournage
d’une expérience concrète de création artistique.

Le programme Banlieuwood permet ainsi aux élèves de vivre l’intégralité du processus créatif d’un film, depuis l’idée initiale jusqu’à la préparation du tournage.

9. Éléments à développer ultérieurement

Certains éléments pourront être précisés dans les versions futures du document :

durée exacte de chaque module
adaptation selon l’âge des élèves
variantes pédagogiques selon les contextes scolaires
intégration de nouveaux modules artistiques (musique, théâtre, écriture).
