# MODULE 8

MODULE 8 — DOCUMENT DE CONCEPTION DÉTAILLÉ

L’équipe — transformer les élèves en équipe de tournage

1. PLACE DU MODULE 8 DANS LA MÉTHODE

Le Module 8 est le dernier module du parcours.

À ce stade, les élèves ont déjà :
construit une histoire
transformé cette histoire en scénario
imaginé la mise en scène
produit un mini storyboard
identifié les scènes à tourner.

Mais il manque encore une étape essentielle.

Un film ne se fait pas seulement avec :
une histoire
un scénario
des images.

Un film se fait avec une équipe.

Le Module 8 sert donc à transformer un groupe d’élèves en :

une équipe de tournage.

2. OBJECTIF PROFOND DU MODULE 8

Le but du Module 8 n’est pas seulement de distribuer des rôles.

Le véritable objectif est :

faire comprendre que le cinéma est un travail collectif où chaque poste a une responsabilité réelle.

Ce module sert à faire vivre une expérience importante :
chacun a un rôle
chacun contribue différemment
chaque rôle est utile au film.

3. POURQUOI LE MODULE 8 COMMENCE PAR UNE PRÉSENTATION DES MÉTIERS

Avant de distribuer les rôles, il est indispensable que les élèves comprennent :

ce que font réellement les différents métiers.

Nous avons validé que cette introduction dure environ 15 minutes.

Mais cette introduction ne doit pas être un cours classique.

Elle doit être interactive et ludique.

4. LE QUIZ DES MÉTIERS

La première partie du module consiste en un quiz très simple.

Les élèves voient :
des images de techniciens
avec le nom du poste.

Exemples :
réalisateur
cadreur
ingénieur son
assistant réalisateur
script
acteur

Les élèves doivent cliquer sur les postes qu’ils pensent connaître.

5. POURQUOI CE QUIZ EST IMPORTANT

Ce quiz sert à deux choses.

1. révéler les idées reçues

Par exemple :

beaucoup pensent que :

le réalisateur tient la caméra.

Le quiz permet de montrer ces croyances.

2. créer de la curiosité

Quand les élèves voient qu’ils se trompent sur certains postes, ils deviennent plus attentifs à l’explication.

6. LE DÉBRIEF DE L’INTERVENANT

Une fois le quiz terminé, l’intervenant fait un débrief très court.

Durée :

5 à 8 minutes maximum.

Le but n’est pas de faire un cours complet.

Le but est :
d’expliquer le rôle de chaque poste
de corriger les idées fausses
de montrer que chaque poste est important.

7. POURQUOI LA PRÉSENTATION RESTE COURTE

Si la présentation dure trop longtemps :
l’attention chute
l’énergie du module disparaît.

Le Module 8 doit rapidement entrer dans :

la répartition des rôles.

8. LA TRANSITION VERS LES RÔLES

Une fois les métiers présentés, l’intervenant explique :

maintenant, vous allez former l’équipe du film.

Mais à ce moment-là intervient un élément central du système Banlieuwood :

le système de points.

9. POURQUOI LE SYSTÈME DE POINTS EXISTE

Le système de points ne sert pas à noter les élèves.

Il sert à :

récompenser l’implication.

Ce point est fondamental.

Le système ne récompense pas :
la qualité
le talent brut
l’aisance orale.

Il récompense :
l’implication
la créativité
la participation
l’investissement dans l’atelier.

10. LES TROIS SOURCES DE POINTS

Nous avons validé trois paramètres.

1. participation aux activités

Les réponses aux exercices peuvent être évaluées sur trois niveaux :
description
description + analyse
description + analyse + interprétation

2. le carnet d’idées

Les élèves peuvent noter :
idées
observations
inspirations.

Ce carnet a un poids important dans le système.

3. les observations de l’intervenant

L’intervenant peut ajouter des tags simples :
très créatif
force de proposition
bonne écoute
perturbateur
très investi.

Ces observations enrichissent la data.

11. POURQUOI LE CLASSEMENT N’EST PAS PUBLIC

Nous avons validé que le classement ne soit pas affiché.

Pourquoi ?

Parce qu’un classement public créerait :
frustration
comparaison
humiliation possible.

Le classement reste invisible.

La seule conséquence visible est :

l’ordre de choix des rôles.

12. LE CHOIX DES RÔLES

Les élèves choisissent les rôles dans l’ordre du classement.

Cela signifie :
le premier choisit en premier
puis le second
puis le troisième.

Mais cela ne veut pas dire que le premier choisira forcément le rôle de réalisateur.

Chacun choisit selon :
ses envies
ses affinités.

13. LE VETO BANLIEUWOOD

Nous avons validé qu’il existe un veto Banlieuwood.

Pourquoi ?

Parce que l’équipe encadrante peut repérer des talents ou des prédispositions.

Exemple :
un élève très à l’aise devant la caméra
un élève qui a une vraie vision visuelle
un élève très organisé.

Dans ces cas-là, Banlieuwood peut orienter un rôle.

Ce veto reste rare, mais il garantit la cohérence du tournage.

14. LES CARTES TALENTS

Le Module 8 introduit aussi un objet symbolique important :

les cartes talents.

Ces cartes sont inspirées :
des cartes Pokémon
ou des cartes FIFA.

15. POURQUOI LES CARTES TALENTS EXISTENT

Ces cartes ont plusieurs fonctions.

1. créer un souvenir

Les élèves repartent avec une carte personnalisée.

2. valoriser chaque élève

Chaque carte met en avant :
une force
une capacité.

3. préparer la cartographie des talents

Les données récoltées permettent de créer une cartographie future.

16. LES CATÉGORIES DE TALENTS

Les talents sont classés en trois grandes familles.

1. jeu / interprétation

élèves à l’aise devant la caméra.

2. mise en scène / image

élèves sensibles au cadrage et à la narration visuelle.

3. technique / organisation

élèves attentifs à la structure et au fonctionnement.

17. POURQUOI IL N’Y A PAS DE FAIBLESSES SUR LES CARTES

Les cartes ne doivent jamais juger.

Elles ne doivent contenir que :
des forces
des axes de progression.

L’objectif est de valoriser.

18. LA MÉTAPHORE DE L’ÉQUIPE DE FOOT

Le système de rôles est inspiré d’une logique d’équipe sportive.

Chaque poste correspond à une fonction dans l’équipe.

Cela aide les élèves à comprendre :

un film fonctionne comme une équipe.

19. LES POSTES POSSIBLES

Les rôles principaux sont :
réalisateur
cadreur
ingénieur son
assistant réalisateur
script
acteurs.

D’autres rôles peuvent exister selon la taille du groupe.

20. LES DOCUMENTS PAR POSTE

Chaque rôle reçoit un document simple.

Ces documents sont générés automatiquement à partir :
du scénario
du découpage
des rôles.

Exemples :

réalisateur

extraits du scénario + intentions.

cadreur

liste des plans + storyboard.

script

continuité.

assistant réalisateur

plan de travail simplifié.

21. POURQUOI CES DOCUMENTS SONT SIMPLES

Ils ne doivent pas ressembler à de vrais documents professionnels.

Ils doivent être :
clairs
visuels
compréhensibles.

22. DIFFÉRENCE FORMULE 2 / FORMULE 3

Formule 2

Les élèves gardent leur rôle pendant tout le tournage.

Pourquoi ?

Parce que les tournages sont courts.

Formule 3

Les élèves observent plusieurs postes.

Ils choisissent :
un poste pour la première demi-journée
un poste pour la seconde.

Les tournages étant plus longs, cela permet une rotation.

23. CE QUE LE MODULE 8 DOIT FAIRE RESSENTIR

Le Module 8 doit créer un moment très particulier.

Les élèves doivent sentir que :

maintenant, le film va vraiment exister.

C’est le moment où le projet devient concret.

24. CE QUE LE MODULE 8 PRODUIT

À la fin du Module 8, le groupe possède :
une équipe de tournage
des rôles attribués
des documents de travail
un film prêt à être tourné.

25. RÉSUMÉ STRATÉGIQUE DU MODULE 8

Si tu dois le résumer à ton associé :

Le Module 8 transforme les élèves en équipe de tournage en distribuant les rôles selon l’implication dans l’atelier et en préparant chacun à sa responsabilité dans la fabrication du film.

26. PLACE DU MODULE 8 DANS L’ENSEMBLE

Si on regarde toute la progression :

Module 1 → regarder
Module 2 → comprendre
Module 3 → imaginer
Module 4 → clarifier
Module 5 → assembler
Module 6 → écrire
Module 7 → visualiser
Module 8 → produire

Le Module 8 est donc le moment où :

le film devient un projet réel.
