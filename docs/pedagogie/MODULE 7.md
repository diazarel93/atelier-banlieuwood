# MODULE 7

BANLIEUWOOD

MODULE 7 — DOCUMENT DE CONCEPTION DÉTAILLÉ

La mise en scène — transformer le scénario en images

1. PLACE DU MODULE 7 DANS LA MÉTHODE

Le Module 7 arrive après le Module 6.

À la fin du Module 6, le groupe possède désormais :
une histoire collective
un scénario renforcé
des scènes identifiées
des personnages définis
des dialogues amorcés
un récit suffisamment stable.

Mais même à ce stade, il manque encore une dimension fondamentale :

le cinéma ne raconte pas seulement une histoire, il la montre.

Le Module 7 est donc le moment où l’on bascule :

du scénario vers la mise en scène.

C’est ici que les élèves comprennent une chose essentielle :

une scène écrite peut être filmée de mille façons différentes.

2. OBJECTIF PROFOND DU MODULE 7

Le but du Module 7 n’est pas d’enseigner la mise en scène de manière académique.

Ce n’est pas :
un cours technique de cinéma,
ni un module d’analyse théorique.

Le véritable objectif est :

faire comprendre aux élèves que raconter au cinéma consiste à choisir quoi montrer et comment le montrer.

C’est la première rencontre des élèves avec la grammaire visuelle du cinéma.

3. LA TRANSITION ENTRE MODULE 6 ET MODULE 7

Le Module 6 a produit un scénario.

Mais ce scénario reste encore une suite de scènes écrites.

Or un film ne se tourne pas en “scènes écrites”.

Il se tourne en :
plans
angles
mouvements
regards
rythme visuel

Le Module 7 sert donc à faire comprendre que :

une scène n’existe au cinéma que lorsqu’on décide où placer la caméra.

4. POURQUOI LE MODULE 7 DOIT ÊTRE TRÈS VISUEL

Nous avons validé un choix très clair :

le Module 7 doit être principalement visuel.

Pourquoi ?

Parce que la mise en scène s’apprend beaucoup plus vite :
en voyant
qu’en lisant une définition

Expliquer à un élève :

“un plan large sert à situer l’espace”

est beaucoup moins efficace que de lui montrer :
une image
puis une autre image
puis leur différence.

Le Module 7 doit donc fonctionner comme une boîte à outils visuelle.

5. POURQUOI ON NE MONTRE QUE QUELQUES TYPES DE PLANS

Dans le cinéma réel, il existe des dizaines de types de plans.

Nous avons volontairement réduit cette complexité.

Pourquoi ?

Parce que l’objectif n’est pas d’enseigner la théorie complète du cinéma.

L’objectif est de transmettre les bases minimales qui permettent de raconter une scène.

Nous avons donc validé une logique simple :

les élèves doivent comprendre 4 types de plans essentiels.

Ces plans couvrent déjà l’essentiel des situations narratives.

6. LES 4 TYPES DE PLANS FONDAMENTAUX

Les quatre plans retenus sont :

1. Plan large

Il sert à situer l’espace.

Il répond à la question :

où sommes-nous ?

Il permet de :
comprendre le lieu
situer les personnages
installer l’ambiance.

2. Plan moyen

Il montre le personnage en situation.

Il sert à :
observer l’action
suivre une interaction
comprendre ce qui se passe.

3. Gros plan

Il montre une émotion ou un détail important.

Il sert à :
capter un regard
montrer un objet clé
accentuer un moment.

4. Plan réaction

Il montre comment un personnage réagit.

Il sert à :
révéler une émotion
créer une tension
donner une information au spectateur.

7. POURQUOI CES 4 PLANS SUFFISENT

Ces quatre plans permettent déjà de construire :
une entrée dans la scène
une action
une émotion
une réaction

Autrement dit :

une mini-structure visuelle complète.

Cela permet aux élèves de comprendre rapidement que :

le cinéma est une succession de points de vue.

8. LE CHOIX VISUEL : APPRENDRE PAR COMPARAISON

Nous avons validé que l’apprentissage se fasse par comparaison d’images.

Exemple :

un écran montre :
une scène filmée en plan large
la même scène en gros plan

Puis on demande :

qu’est-ce que ça change ?

Pourquoi ce système est puissant ?

Parce que les élèves découvrent eux-mêmes la différence.

On ne leur donne pas une règle abstraite.

Ils la ressentent.

9. POURQUOI LE MODULE 7 DOIT RESTER TRÈS COURT SUR LA THÉORIE

Nous avons clairement validé que la partie “explication” devait rester courte.

Pourquoi ?

Parce que ce module doit très vite basculer vers :

l’application.

L’objectif n’est pas que les élèves retiennent une leçon.

L’objectif est qu’ils comprennent immédiatement :

comment transformer une scène en images.

10. LE CHOIX DES SCÈNES À TRAVAILLER

Le Module 7 ne traite pas tout le scénario.

Ce serait trop long et inutile.

Nous avons validé que seules quelques scènes clés soient travaillées.

Minimum :
2 scènes obligatoires

Possibilité :
3 scènes si nécessaire

Ces scènes doivent être :

1️⃣ une scène de discussion
2️⃣ une scène d’action ou de tension

Pourquoi ce choix ?

Parce que ces deux types de scènes couvrent déjà la majorité des situations cinématographiques.

11. POURQUOI UNE SCÈNE DE DISCUSSION EST IMPORTANTE

Une scène de discussion est intéressante parce que :

elle oblige à réfléchir à :
qui regarde qui
quand couper
quelle réaction montrer
quel détail filmer

Les élèves comprennent alors que même une simple discussion peut devenir cinématographique.

12. POURQUOI UNE SCÈNE D’ACTION OU DE TENSION EST IMPORTANTE

Une scène d’action ou de tension oblige à réfléchir à :
le rythme
les regards
les mouvements
la surprise

Elle montre que le montage et les plans peuvent créer :

de l’énergie.

C’est une excellente porte d’entrée vers le langage du cinéma.

13. LE MINI-DÉCOUPAGE

Pour chaque scène, les élèves doivent produire un mini découpage.

Mais attention :

nous avons volontairement choisi un découpage très simplifié.

Pourquoi ?

Parce que le but n’est pas de créer un vrai découpage technique professionnel.

Le but est que les élèves comprennent :

une scène est une suite de plans choisis.

Le mini découpage doit donc contenir :
le type de plan
ce que l’on voit
l’intention du plan.

14. POURQUOI LE DÉCOUPAGE N’EST PAS TECHNIQUE

Nous avons volontairement supprimé toute complexité technique :
pas de focale
pas d’axe caméra compliqué
pas de jargon.

Pourquoi ?

Parce que l’important est de comprendre :

la logique narrative du plan.

15. LA LOGIQUE DU STORYBOARD SIMPLIFIÉ

Le Module 7 introduit également une forme de storyboard simplifié.

Cela signifie :

les plans peuvent être représentés par :
des images générées
ou des illustrations simples.

Pourquoi ce choix est très fort ?

Parce que les élèves peuvent voir le film avant de le tourner.

Ils commencent à comprendre :

qu’un film se prépare visuellement.

16. POURQUOI LES IMAGES SONT GÉNÉRÉES ENTRE LES SÉANCES

Nous avons validé un point technique important.

Les images utilisées pour le storyboard ne sont pas générées pendant le module.

Elles sont générées en inter-séance.

Pourquoi ?

Parce que la génération d’image en direct :
ralentirait énormément le module
créerait des temps morts
rendrait l’atelier fragile techniquement.

Le Module 7 doit être fluide.

Les images doivent donc déjà être prêtes.

17. CE QUE LE MODULE 7 DOIT FAIRE RESSENTIR

Le Module 7 doit produire une sensation très particulière :

les élèves doivent commencer à voir leur film dans leur tête.

Ce moment est extrêmement important.

C’est souvent là que le projet devient réel pour eux.

Ils ne parlent plus seulement d’une histoire.

Ils parlent d’images.

18. LE RÔLE DE L’INTERVENANT DANS CE MODULE

L’intervenant doit :
expliquer les bases visuelles
encourager les propositions
aider à clarifier les plans
vérifier la cohérence des scènes.

Il ne doit pas :
imposer un découpage
corriger comme un professeur
chercher la perfection.

Le Module 7 est un terrain d’exploration visuelle.

19. CE QUE LE MODULE 7 DOIT PRODUIRE À LA FIN

À la fin du Module 7, le groupe doit avoir :
un mini storyboard
un mini découpage de scènes clés
une compréhension de la grammaire visuelle
une première vision concrète du film.

Autrement dit :

le film existe désormais dans les images.

20. INTERSÉANCE MODULE 7 → MODULE 8

Cette interséance prépare le dernier module.

Elle sert à transformer :
le scénario
le storyboard
les scènes choisies

en documents de tournage simplifiés.

Ces documents seront adaptés aux rôles.

Exemples :
fiche réalisateur
fiche chef opérateur
fiche script
fiche assistant réal
fiche acteurs

Chaque poste reçoit donc :

une version simplifiée des informations dont il a besoin.

21. POURQUOI CELA PRÉPARE LE MODULE 8

Le Module 8 n’est pas un module d’écriture.

C’est un module de préparation du tournage.

Les élèves doivent donc arriver avec :
une histoire claire
des scènes définies
une vision visuelle
et des rôles.

Le Module 7 permet précisément cette transition.

22. LES PIÈGES DU MODULE 7

Le Module 7 peut échouer si :

piège 1

on le transforme en cours de cinéma.

piège 2

on donne trop de théorie.

piège 3

on complexifie les types de plans.

piège 4

on veut un découpage parfait.

piège 5

on oublie que le but est de voir le film.

23. CE QUE LE MODULE 7 APPREND EN RÉALITÉ AUX ÉLÈVES

Sans qu’ils s’en rendent compte, les élèves apprennent :
que le cinéma est un langage
qu’une scène peut être racontée de plusieurs façons
que la caméra est un choix
que l’image raconte autant que les mots.

C’est souvent à ce moment que les élèves comprennent vraiment :

ce qu’est le cinéma.

24. RÉSUMÉ STRATÉGIQUE DU MODULE 7

Si tu dois le résumer à ton associé en une phrase :

Le Module 7 transforme le scénario écrit au Module 6 en vision visuelle du film, en apprenant aux élèves les bases du langage cinématographique et en produisant un mini storyboard des scènes clés.

Et si tu veux résumer sa place dans la progression :

c’est le module où l’histoire devient enfin cinéma.
